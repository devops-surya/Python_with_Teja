Store POS - Python Terminal Billing Project

A complete beginner-friendly guide to a Point of Sale terminal project. The app stores inventory in memory, adds sale items to a cart, calculates the bill, and clears the cart after payment.

Pure Python Terminal App Inventory Cart Billing No Database

Project Snapshot

InventoryProducts are stored as dictionaries inside the ITEMS list.
CartSelected item ids and quantities are stored inside the CART dictionary.
BillingThe bill is calculated by multiplying price and quantity for each cart item.

Folder Structure

TERMINAL
POS/ | |-- pos.py # Main file: menus and user flow |-- inventory.py # Inventory list and item functions |-- sales.py # Cart and billing functions |-- Readme.md # Markdown explanation |-- README.html # Visual HTML explanation

Data Design

The inventory starts empty. Each new item becomes a dictionary inside the ITEMS list.

PYTHON
# inventory.py ITEMS = [] # example item after user adds it { "id": 1, "name": "Pendrive", "description": "64GB USB storage device", "price": 599.0, "quantity": 30 } # sales.py CART = { 1: 2, 2: 1 }

Function Reference

FileFunctionWhat it does
pos.pymain_menu()Shows the main menu.
pos.pyinventory_menu()Takes product input and adds inventory.
pos.pysale_menu()Takes sale item id and quantity.
inventory.pyadd_item()Adds a dictionary to ITEMS.
inventory.pyget_item()Returns item details by id.
inventory.pyfind_item()Finds an item index.
sales.pyadd_item_to_cart()Adds item id and quantity to cart.
sales.pygenerate_bill()Calculates total bill.
sales.pyfinish_transaction()Clears the cart after payment.
