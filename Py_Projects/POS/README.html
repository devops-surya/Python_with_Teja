Python Terminal Project

LT StoreDesk POS

A polished guide for explaining a beginner Point of Sale project before coding it. The project keeps the logic small, visual, and classroom friendly: add products, create a cart, calculate a bill, and complete the sale.

No database Pure Python Terminal UI Inventory + Billing

Project Snapshot

Inventory Stores products as dictionaries inside a list and lets the user add new stock items.
Sales Cart Stores selected product ids with quantities for the current customer transaction.
Billing Reads product prices from inventory, multiplies by quantity, and prints the final amount.

Actual Function Reference

FileFunctionRole
pos.pymain_menu()Shows the main menu and routes choices.
pos.pyinventory_menu()Collects new product details.
pos.pysale_menu()Handles sale item selection and payment confirmation.
inventory.pyadd_item()Adds product data to inventory.
inventory.pyget_item()Returns product details by id.
sales.pyadd_item_to_cart()Adds selected id and quantity to the cart.
sales.pygenerate_bill()Calculates the final bill amount.
sales.pyfinish_transaction()Clears the cart after payment.

Teaching Order

BUILD PLAN
1. Explain the shop billing problem 2. Design one product dictionary 3. Store products in a list 4. Print the inventory 5. Add new inventory items 6. Create the customer cart 7. Add product id and quantity to cart 8. Calculate price x quantity 9. Connect everything through the main menu 10. Run a complete billing demo
This version is intentionally small. Good next upgrades are stock validation, quantity reduction after billing, formatted receipts, discounts, file storage, and sales history.
